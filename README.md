# A/B Test Statistics Dashboard

An interactive line chart visualization for A/B test conversion rate analysis, built with React, TypeScript, and Recharts.

__https://clericlvl2.github.io/ab-line-chart/__

## Features

### Core
- Interactive line chart for conversion rate
- Hover tooltips
- Data-variation selector (min 1 selection)
- Day/Week time range toggle
- Responsive design (671px - 1300px)

### Additional
- Line style selector (Line, Smooth Line, Area)
- Light/Dark theme toggle
- Automatic axis scaling

## Technologies

- React 19
- TypeScript
- Recharts
- Vite
- CSS Modules
- Biome

## Installation

1. Clone the repository:
```bash
git clone https://github.com/clericlvl2/ab-line-chart.git
```

2. Navigate to the project folder:
```bash
cd ab-line-chart
```

3. Install dependencies and start the development server:
```bash
npm install
npm run dev
```

## Usage

- **Variations Selector**: Toggle which test variations to display on the chart
- **Time Range**: Switch between daily and weekly aggregated views
- **Chart Style**: Choose between Line, Smooth Line, or Area visualization
- **Theme**: Toggle between light and dark modes
- **Hover**: Mouse over the chart to see detailed metrics for specific dates
